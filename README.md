# BiznesElektroniczny
Projekt na przedmiot Biznes Elektroniczny PG

Opis projektu 

Celem projektu było przygotowanie umiejętności praktycznych z zakresu stosowania, konfiguracji i wdrożenia oprogramowania Open Source dedykowanego do tworzenia sklepów internetowych. W ramach zajęć mieliśmy możliwość korzystać z oprogramowania PrestaShop i dostosowywać go do określonych wymagań.  
Projekt miał za zadanie przybliżyć nam warunki pracy programisty/programistki zbliżone do rzeczywistych. Ukazując proces od projektu rozwiązania, poprzez dostosowanie produktu dla klienta, współpracę w zespole programistycznym oraz zadania związane z końcowym przygotowaniem produktu do uruchomienia produkcyjnego.


Wykorzystane narzędzia 
* Repozytorium kodu źródłowego - https://github.com 
* Narzędzie do zarządzania zmianami w kodzie - https://git-scm.com
* Platforma sklepu internetowego - Prestashop 1.7.8 - https://github.com/PrestaShop/PrestaShop/tree/1.7.8.x
* Dowolny edytor kodu źródłowego np. NeoVim/SublimeText/Visual Code - https://code.visualstudio.com
* System konteneryzacji aplikacji - Docker https://docker.com
* System do zarządzania kompozycją kontenerów - https://docs.docker.com/compose/
* Biblioteka do budowy testów UI - https://www.selenium.dev/documentation/en/
* Podczas konfiguracji środowiska wirtualnego będą przydatne dodatkowe pakiety np.: apache (httpd2), mysql (mariadb), git, php, php-mysql, php-pdo, i inne wskazane przez platformę do tworzenia sklepów internetowych (skrypt instalacyjny wskaże brakujące biblioteki).
